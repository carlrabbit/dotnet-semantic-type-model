# Diagnostics Preview Status

## Stability

Diagnostic IDs and exact messages should be treated as preview/unstable in the current milestone set.

## Usage Guidance

When consuming diagnostics:

- treat IDs as subject to change across pre-release milestones;
- prefer severity and category handling over exact message text;
- validate behavior against current release notes.
